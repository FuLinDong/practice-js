1、函数传参 一个参数

定义函数的参数为color，变量adiv的样式调用参数color，onclick事件把背景颜色的值传给参数color。

2、函数传参 两个参数

定义函数的参数为name和value，变量bdiv的样式的属性名用参数name代替， 属性值用参数value代替，onclick事件把样式的属性名和值分别传给参数name和value。

3、函数传参 参数加变量

定义函数的参数为name，定义变量a为参数name，两个onclick事件分别绑定标签属性调用参数的值。(暂时不知道为什么要把参数定义成变量，但是这样也行的通。)

记录

函数传参：参数就是占位符
方括号操作属性：当要修改的属性不固定的时候，就用方括号来操作。只要是可以用.的地方，都可以用方括号来表示。
通过style，操作的只能是行间样式。无论是读还是写。
样式的优先级：*（通配符）标签 class Id 行间，如果先修改了行间，再修改class就会失败。因为优先级。所以对于一个元素，要么只修改行间，要么只修改class，不能混着来。
